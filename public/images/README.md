# 化妆品图片占位说明

由于项目需要50张化妆品图片，您需要将图片放置在此目录下。

## 图片命名规范

- q1.jpg - q50.jpg
- 建议尺寸：800x800px
- 建议大小：每张 < 200KB
- 格式：JPG 或 PNG

## 图片来源建议

1. **无版权图片网站**（推荐）
   - Unsplash (https://unsplash.com)
   - Pexels (https://pexels.com)
   - Pixabay (https://pixabay.com)

2. **自行拍摄**
   - 拍摄您自己的化妆品

3. **购买授权图片**
   - 从图库网站购买

## 临时占位

在开发阶段，您可以使用占位图片服务：
- https://via.placeholder.com/800x800
- https://picsum.photos/800/800

## 注意事项

- 确保图片清晰可见
- 避免使用有版权争议的图片
- 图片内容应与题目描述相符
