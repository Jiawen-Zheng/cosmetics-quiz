# 手动修改和上传指南

## 📍 文件位置

### 1. 问题库文件
**位置：** [`cosmetics-quiz/src/data/questions.ts`](../src/data/questions.ts)

这个文件包含所有的测验题目，每个题目的结构如下：
```typescript
{
  id: 1,
  question: "题目文字",
  options: ["选项A", "选项B", "选项C", "选项D"],
  correctAnswer: 0, // 正确答案的索引（0=A, 1=B, 2=C, 3=D）
  explanation: "答案解释",
  image: "/images/product-1.jpg", // 对应的图片路径
  difficulty: "easy" // 难度：easy/medium/hard
}
```

### 2. 图片文件夹
**位置：** [`cosmetics-quiz/public/images/`](../public/images/)

- 当前有 50 张产品图片（product-1.jpg 到 product-50.png）
- 支持的格式：`.jpg`、`.png`、`.webp`
- 建议尺寸：800x800 像素左右
- 建议大小：每张图片不超过 200KB

---

## 🔧 如何修改

### 修改问题库
1. 用 VSCode 打开 [`cosmetics-quiz/src/data/questions.ts`](../src/data/questions.ts)
2. 找到你要修改的题目（通过 id 或题目内容）
3. 修改题目文字、选项、正确答案或解释
4. 保存文件（Ctrl+S）

**示例修改：**
```typescript
// 修改前
{
  id: 1,
  question: "旧题目",
  options: ["A", "B", "C", "D"],
  correctAnswer: 0,
  explanation: "旧解释",
  image: "/images/product-1.jpg",
  difficulty: "easy"
}

// 修改后
{
  id: 1,
  question: "新题目",
  options: ["新A", "新B", "新C", "新D"],
  correctAnswer: 2, // 改为C选项
  explanation: "新解释",
  image: "/images/product-1.jpg",
  difficulty: "medium" // 改为中等难度
}
```

### 替换图片
1. 准备好你的新图片
2. 重命名为 `product-X.jpg` 或 `product-X.png`（X 是题目编号）
3. 复制到 [`cosmetics-quiz/public/images/`](../public/images/) 文件夹
4. 覆盖原有的同名文件

**注意事项：**
- 保持文件名格式一致（product-1.jpg, product-2.png 等）
- 图片尽量优化到 200KB 以内
- 建议使用正方形图片（如 800x800）

## 📤 如何手动上传到 GitHub

### 方法一：使用 VSCode 内置 Git（推荐）

#### 步骤 1：查看修改
1. 点击 VSCode 左侧的"源代码管理"图标（或按 `Ctrl+Shift+G`）
2. 你会看到所有修改过的文件列表

#### 步骤 2：暂存修改
1. 在修改的文件上点击 `+` 号（暂存更改）
2. 或点击顶部的 `+` 号暂存所有更改

#### 步骤 3：提交修改
1. 在顶部的消息框中输入提交说明，例如：
   - `更新问题库内容`
   - `替换产品图片`
   - `修改题目和答案`
2. 点击 `✓` 提交按钮（或按 `Ctrl+Enter`）

#### 步骤 4：推送到 GitHub
1. 点击顶部的 `...` 菜单
2. 选择"推送"（Push）
3. 等待上传完成

---

### 方法二：使用命令行

#### 步骤 1：打开终端
在 VSCode 中按 `` Ctrl+` `` 打开终端

#### 步骤 2：执行以下命令
```bash
# 1. 查看修改了哪些文件
git status

# 2. 添加所有修改的文件
git add .

# 3. 提交修改（引号内写你的修改说明）
git commit -m "更新问题库和图片"

# 4. 推送到 GitHub
git push
```

---

### 方法三：使用 GitHub Desktop（最简单）

如果你安装了 GitHub Desktop：

1. 打开 GitHub Desktop
2. 选择 `cosmetics-quiz` 仓库
3. 左侧会显示所有修改的文件
4. 在左下角输入提交说明
5. 点击"Commit to main"按钮
6. 点击顶部的"Push origin"按钮

---

## 🚀 推送后自动部署

推送到 GitHub 后，Vercel 会自动检测到更改并重新部署：

1. **自动触发**：推送后 1-2 分钟内开始部署
2. **查看进度**：访问 [Vercel Dashboard](https://vercel.com/dashboard)
3. **部署时间**：通常 2-5 分钟完成
4. **自动生效**：部署成功后，网站自动更新

## ⚠️ 常见问题

### 1. 推送失败：需要登录
**解决方法：**
```bash
# 配置 Git 用户信息
git config --global user.name "你的GitHub用户名"
git config --global user.email "你的GitHub邮箱"
```

### 2. 推送失败：权限问题
**解决方法：**
- 确保你已登录 GitHub 账号
- 在 VSCode 中：点击左下角账号图标，选择"登录 GitHub"

### 3. 图片不显示
**检查清单：**
- ✅ 图片文件名是否正确（product-X.jpg）
- ✅ 图片是否在 `public/images/` 文件夹中
- ✅ questions.ts 中的路径是否正确（`/images/product-X.jpg`）
- ✅ 是否已推送到 GitHub 并重新部署

### 4. 修改后网站没更新
**解决方法：**
1. 确认已成功推送到 GitHub
2. 访问 Vercel Dashboard 查看部署状态
3. 清除浏览器缓存（Ctrl+Shift+R 强制刷新）
