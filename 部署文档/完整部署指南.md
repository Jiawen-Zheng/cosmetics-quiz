## 第一步：安装Git（3分钟）

### 什么是Git？
Git是一个工具，用来上传你的代码到GitHub。

### 操作步骤

**1.1 下载Git**

1. 打开浏览器
2. 访问：https://git-scm.com/download/win
3. 页面会自动开始下载
4. 等待下载完成（文件名类似：Git-2.43.0-64-bit.exe）

**1.2 安装Git**

1. 双击下载的文件
2. 看到安装界面，点击【Next】
3. 选择安装位置（建议默认位置）
4. 一直点击【Next】，使用默认选项
5. 最后点击【Install】
6. 安装完成，点击【Finish】

**1.3 验证安装**

1. 按键盘上的 `Win + R`
2. 输入：`cmd`
3. 按回车，打开黑色窗口
4. 输入：`git --version`
5. 按回车
6. 如果显示版本号（如：git version 2.43.0），说明安装成功！

**✅ 检查点：**
- [ ] Git已下载
- [ ] Git已安装
- [ ] 能看到Git版本号

## 第二步：注册GitHub（2分钟）

3. 点击【Continue】
4. 完成验证（可能需要点击图片或输入验证码）
5. 点击【Create account】

**2.3 验证邮箱**

1. 打开你的邮箱
2. 找到GitHub发来的邮件
3. 点击邮件中的【Verify email address】
4. 验证成功！

**2.4 完成设置**

1. 选择【Skip personalization】（跳过个性化设置）
2. 进入GitHub主页

**✅ 检查点：**
- [ ] GitHub账号已注册

## 第四步：上传代码（3分钟）

### 现在要做什么？
把你电脑上的项目代码上传到GitHub仓库。

### 操作步骤

**4.1 打开命令提示符**

1. 按键盘 `Win + R`
2. 输入：`cmd`
3. 按回车
4. 看到黑色窗口

**4.2 进入项目目录**

在黑色窗口中，输入以下命令（一行一行输入，每行输入后按回车）：

```bash
cd /d "E:\JIA project\03 cosmetics\cosmetics-quiz"
```

**注意：** 如果你的项目在其他位置，请修改路径！

按回车后，你会看到路径变成了项目目录。

**4.3 配置Git（只需要做一次）**

输入以下命令（替换成你自己的信息）：

```bash
git config --global user.name "Jiawen-Zheng"
```
按回车

```bash
git config --global user.email "Zjiawen0906@163.com"
```
按回车



**4.4 初始化Git仓库**

输入：
```bash
git init
```
按回车

看到提示：`Initialized empty Git repository`，说明成功！

**4.5 添加所有文件**

输入：
```bash
git add .
```
按回车

**注意：** `add` 后面有一个空格和一个点（.）

**4.6 提交文件**

输入：
```bash
git commit -m "Initial commit"
```
按回车

会看到很多文件被提交的信息。

**4.7 关联远程仓库**

输入（替换成你的GitHub用户名）：
```bash

```
按回车

**示例：**
```bash
git remote add origin https://github.com/zhangsan/cosmetics-quiz.git
```

**4.8 推送到GitHub**

输入：
```bash

```
按回车

然后输入：
```bash
git push -u origin main
```
按回车

**4.9 输入GitHub账号密码**

1. 选择【Sign in with your browser】（用浏览器登录）

3. 输入你的GitHub用户名和密码
4. 点击【Sign in】
5. 点击【Authorize】（授权）
6. 回到命令提示符，等待上传完成

**看到类似这样的提示，说明成功：**

- [ ] 命令都执行成功

## 第六步：配置Cloudflare DNS（5分钟）

### 什么是Cloudflare？
Cloudflare提供CDN加速服务，让你的网站在全球都能快速访问。

### 前提条件

你的域名 `linggxz.online` 应该已经：
- ✅ 在Namesilo购买
- ✅ DNS服务器已指向Cloudflare
- ✅ 在Cloudflare中已添加

如果还没有完成，请先参考之前的域名配置文档。

### 操作步骤

**6.1 登录Cloudflare**

1. 访问：https://www.cloudflare.com
2. 登录你的账号
3. 看到域名列表

**6.2 选择域名**

1. 点击 `linggxz.online` 域名
2. 进入域名管理页面

**6.3 添加DNS记录**

1. 点击【DNS】标签
2. 点击【Add record】按钮
3. 填写DNS记录信息：

```
类型（Type）：CNAME

名称（Name）：cosmetics
（这将创建 cosmetics.linggxz.online 子域名）

目标（Target）：cname.vercel-dns.com

代理状态（Proxy status）：已代理（橙色云朵）✅
（点击云朵图标，确保是橙色的）

TTL：Auto（自动）
```

4. 点击【Save】保存

**6.4 确认记录已添加**

在DNS记录列表中应该看到：

```
类型    名称         目标                    代理状态    TTL
CNAME   cosmetics   cname.vercel-dns.com    已代理🟠    Auto
```

**✅ 检查点：**
- [ ] 已登录Cloudflare
- [ ] 已选择linggxz.online域名
- [ ] 已添加CNAME记录
- [ ] 代理状态是橙色云朵
- [ ] 记录已保存

## 第七步：绑定域名到Vercel（5分钟）

### 现在要做什么？
告诉Vercel使用你的自定义域名。

### 操作步骤

**7.1 进入Vercel项目**

1. 访问：https://vercel.com
2. 登录你的账号
3. 点击 `cosmetics-quiz` 项目

**7.2 进入域名设置**

1. 点击【Settings】（设置）
2. 点击左侧菜单的【Domains】（域名）

**7.3 添加自定义域名**

1. 在输入框中输入：
   ```
   cosmetics.linggxz.online
   ```

2. 点击【Add】按钮

**7.4 等待验证**

Vercel会自动验证域名配置：

1. **验证过程：**
   - Vercel检查DNS记录
   - 确认CNAME指向正确
   - 生成SSL证书

2. **等待时间：**
   - 通常5-10分钟
   - 最长可能需要30分钟

3. **验证成功标志：**
   - 域名状态显示：✅ Valid Configuration
   - 或显示：✅ Active

**如果验证失败：**
1. 等待10分钟后刷新页面
2. 检查Cloudflare的DNS记录是否正确
3. 确认代理状态已开启（橙色云朵）

**✅ 检查点：**
- [ ] 已进入Vercel项目设置
- [ ] 已添加域名 cosmetics.linggxz.online
- [ ] 域名验证成功
- [ ] 状态显示为 Valid Configuration

## ❓ 常见问题

### Q1：DNS一直不生效怎么办？

**A：** 继续等待
- DNS在中国的传播可能需要24-48小时
- 这是正常现象
- 可以先用Vercel域名测试功能

### Q2：Vercel验证域名失败？

**A：** 检查以下几点
1. DNS记录是否正确：
   - 类型：CNAME
   - 名称：cosmetics
   - 目标：cname.vercel-dns.com
2. 代理状态是否开启（橙色云朵）
3. 等待10分钟后再试

### Q3：网站打不开？

**A：** 按以下步骤排查
1. 检查DNS是否生效（whatsmydns.net）
2. 检查Cloudflare DNS记录是否正确
3. 检查Vercel域名状态是否为 Valid
4. 清除浏览器缓存
5. 等待更长时间

### Q4：访问速度慢？

**A：** 检查以下几点
1. 确认Cloudflare代理已开启（橙色云朵）
2. 检查网络环境
3. 尝试使用手机4G/5G网络
4. 优化网站资源（压缩图片等）

### Q5：如何修改题目？

**A：** 修改题库文件
1. 打开项目文件夹
2. 找到 `src/data/questions.ts` 文件
3. 修改题目内容
4. 保存后按照"如何更新网站"的步骤上传

### Q6：如何替换化妆品图片？

**A：** 替换图片文件
1. 将新图片放入 `public/images/` 文件夹
2. 命名为 product-1.jpg, product-2.jpg 等
3. 保存后按照"如何更新网站"的步骤上传
